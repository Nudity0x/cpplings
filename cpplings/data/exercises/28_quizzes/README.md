# Quizzes

No new concepts here — these quizzes mix everything you've learned so far into
slightly larger programs. Each one leans on several earlier sections at once:
you might combine a class with a container, or an algorithm with a lambda and some
error handling.

Treat them as checkpoints. If a quiz makes you reach back for a concept you're
shaky on, that's a signal to revisit that section's exercises and README. There's
no trick beyond applying the fundamentals together — read the comments, work out
what the program is meant to do, and make it do that.
