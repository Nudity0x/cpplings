# Intro

Welcome to cpplings! These first two exercises aren't really about C++ at all —
they're about the workflow you'll use for every exercise that follows.

Each exercise is a small program that's broken in some way: it won't compile, it
fails a test, or it still has the `// I AM NOT DONE` marker at the top. Your job
is to fix it, remove that marker, and save. The watch loop rechecks the file the
moment you save and moves you along when it passes.

Get comfortable with the loop here: read the comments, make the smallest change
that satisfies them, delete `// I AM NOT DONE`, and save.
